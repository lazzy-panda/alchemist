# Сопоставление: дерево UI → RPG UI-кит (папка Design)

Каждый элемент интерфейса сопоставлен с наиболее подходящим элементом из китов
(`magnific__-ui-kit-rpg` — листы 51936/51937/51940/51943, обзор 51934, панели 5818c9e0)
и перенесён в вёрстку/стили. Дизайн-язык кита: **каменные рамы + парчмент**, глянцевые
«леденцовые» кнопки, тёмные тонкие бордеры в цвет, верхний блик + 3D нижняя кромка.

## Целевая палитра (из кита)
```
Камень: stoneLight #C4BFB2 · stoneMid #9D9888 · stoneDark #6E6A5B · stoneLine #4A463B · stoneEdge #DAD5C6
Кнопки-леденцы (верх→низ / бордер):
  green  #86C760→#4E9B3A / #2E6B22     (основная: OK, BUY, UPGRADE, BACK, CLAIM, Сохранить)
  red    #E8694E→#C23725 / #7C1E10     (опасность/закрытие/NO/SELL)
  gold   #F4C24A→#E0902A / #9A6212     (акцент/TRACK/+ Новая/Завершить)
  blue   #5FB8E8→#2F86C0 / #1C5C8A     (инфо/MAP/вторичные действия)
  stone  #C4BFB2→#9D9888 / #4A463B     (нейтральная: EXIT/Назад/Сброс)
Бары: трек тёмный #3a2f22 + бордер stoneLine; заливка глянцевая (HP красная, Ци бирюза, XP зелёная)
```
