# Alchemist · 修真之路

RPG-трекер созерцательных и телесных практик (медитация, цигун, Чжан Чжуан) в эстетике
Mists of Pandaria / древнего Китая. Практики питают характеристики, ресурсы (HP/Ци),
перки и реликвии персонажа.

Собрано на **React Native (Expo)** один в один по спецификации «Путь культивации» и
анимированному прототипу. Работает на web, iOS и Android из одной кодовой базы.

## Экраны
- **Сегодня** — чек-лист практик дня, состояние персонажа, HP/Ци, стрик
- **Персонаж** — радар-мандала из 7 характеристик, ресурсы, перки, реликвии
- **Практики** — библиотека практик по категориям, редактор практики
- **Дневник** — шестиразовый этический дневник (по Геше Майклу Роучу)
- **Журнал** — тепловая карта, графики роста, сводка периода

Плюс оверлеи: деталь практики с круговым таймером, повышение ступени, детали дня.

## Авторизация
Локальная регистрация / вход с сохранением сессии (AsyncStorage). API (`register` /
`login` / `signOut`) спроектирован под лёгкую замену на реальный бэкенд.

## Запуск
```bash
npm install
npx expo start            # выбрать платформу (w — web)
```

## Сборка web и деплой
```bash
npx expo export --platform web   # -> dist/
```
Деплой статики из `dist/` на любой бесплатный хостинг (GitHub Pages / Netlify / Vercel).
